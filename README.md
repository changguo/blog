## HTTP

> HTTP 协议，是 TCP/IP 协议的应用层协议。规定了客户端与服务器端的传输规则。

一般来说，从浏览器输入一个 URL 会执行以下步骤：

- 浏览器解析出主机名；
- 浏览器查询这个主机名的 ip 地址；
- 浏览器获得端口号；
- 建立到 Web 服务器的 TCP 连接。
- 通过这条连接，发送一条请求报文。
- 浏览器从服务器读取响应报文。
- 浏览器关闭连接。
